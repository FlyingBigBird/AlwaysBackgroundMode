# AlwaysBackgroundMode
应用无限后台模式

后台模式一：
  申请后台程序执行时间,以前能申请十分钟，现在最多180s,用处不大...

后台模式二：
  audio后台，无限静音播放音频，会耗电的说...
 
后台模式三：
  Location Always Usage Description 总是允许使用位置信息,后台一直刷新位置信息
  十分耗用资源而且手机会提示XXXapp一直在使用您的位置，耗费资源较大...

后台模式四：
  voip，网络电话，直播等一直加载网络上数据，看例如后台看直播、打语音电话,这个是很合理的...
  
总结：
  能不用最好不要一直使用后台模式，用户体验不佳不说还损耗电池，有违苹果产品设计的初衷...
